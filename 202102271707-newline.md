# Newline

#it #controlcharacters

Caractère de contrôle marquant la fin d'une ligne dans un fichier.

## Spécifité par OS

| OS                       | Abréviation | Séquence d'échappement |
| ------------------------ | ----------- | ---------------------- |
| Unix-like (Linux, macOS) | LF[^1]          | `\n`                   |
| Windows                  | CR[^2] LF       | `\r\n`                 |

[^1]: Line Feed
[^2]: Carriage Return
